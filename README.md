## 🏢 Project Overview

Influencer Gear is a modern and responsive product showcase website designed for influencers and content creators.  
The website presents high-quality camera gear and accessories using a clean, structured, and visually appealing layout built with HTML and CSS.

This project focuses on product presentation, responsive UI design, and a smooth user experience.

---

## ✨ Key Features

- Fully Responsive Design
- Modern Product Showcase Layout
- Hero Banner Section
- Popular Product Cards Grid
- Featured Gear Section
- Clean and Minimal UI Design
- Pure HTML5 & CSS3 Project
